# NagrikPulse — static prototype

A compact, static prototype for a citizen reporting portal named “NagrikPulse”. This repository contains the original HTML you provided (nagrikpulse.html) and a small wrapper so it can be hosted easily.

This is a static, front-end-only demo intended for local preview, quick hosting (GitHub Pages / Netlify) and as a starting point for further development.

Features
- Clean, responsive layout and components (no build step required).
- Simple demo authentication flow (OTP simulated on the client).
- Interactive report list, filters, map pins and a grievance form.
- Minimal, friendly docs so the repo looks like a human-made project.

How to preview locally
1. Open `nagrikpulse.html` directly in a browser (double-click the file).
2. Or run a simple local server (recommended) from the project folder:
   - Python 3: `python -m http.server 8080`
   - Then open http://localhost:8080/ in your browser.

Deploy to Netlify / Vercel
- Connect the repository to Netlify or Vercel. No build step required.
